sjcx-tipbot
===========

Tip anyone on twitter or IRC with SJCX the coin used in http://storj.io/

###Dependencies
1. MongoDB
2. Node.js
3. bitcoind
4. counterpartyd

###Install
```
git clone https://github.com/tipsjcx/sjcx-tipbot.git
cd sjcx-tipbot
npm install
Enter your twitter api tokens in twitterbot.js
Change the nick name in ircbot.js
node app.js
```
